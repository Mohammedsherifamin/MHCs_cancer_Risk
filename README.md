# Mental health conditions and incident cancer

This repository contains the analysis code for the published article:
M. S.Amin, S.Parra-Soto, Z.Zhou, et al., “Mental Health Conditions and Incident Cancer: A Prospective Cohort Study of 402,255 UK Biobank Participants,” International Journal of Cancer (2026): 1–12, https://doi.org/10.1002/ijc.70527.

## Code structure

The analysis code is organised as follows:

1. `1.data_preparation.R` prepares and cleans the data.
2. `2.descriptives.R` produces descriptive tables.
3. `3.main_analysis.R` runs the main Cox regression models.
4. `4.main_analysis_forest.R` produces the main forest plots.
5. `5.sensitivity.R` runs the PHQ-4 sensitivity analysis.
6. `6.evalue.R` calculates E-values.

An earlier version of this work was posted as a medRxiv preprint. This repository has been updated to align with the final published article.
